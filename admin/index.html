GitHub Token Setup
Connect to GitHub API to save your changes directly to your repository.
Setup Instructions
1
Go to github.com/settings/tokens/new (Classic Tokens)
2
Set a note (e.g. portfolio-cms), expiration = No expiration
3
Check repo scope (full repository access). Click Generate token.
4
Copy the token and paste it below. It's stored only in your browser's localStorage — never uploaded.
